## How to Use

### GitHub Copilot CLI

```bash
copilot

# Review a PR
# → Automatically uses pr agent

# Write tests
# → Automatically uses write-tests-agent

# Triage issues
# → Uses issue-triage skill
```

### GitHub Copilot Coding Agent (Remote)

The `copilot-setup-steps.yml` workflow pre-builds the repo so Copilot can work on it remotely.

### Daily Automation

- **Similar Issues** — New issues are automatically checked for duplicates
- **Heat Sensor** — Comments are monitored for heated language

## Customization

### Adding Domain Instructions

Create a new file in `.github/instructions/`:

```markdown
---
applyTo:
  - "src/MyArea/**"
---

# MyArea Guidelines
[Your instructions here]
```

### Adding Skills

Create a new directory in `.github/skills/`:

```
.github/skills/my-skill/
└── SKILL.md
```

### Improving Instructions

After any PR, use the learn-from-pr skill/agent to extract lessons and improve the AI infrastructure.

## Generated By

This AI infrastructure was generated by [ai-native-dotnet-bootstrap](https://github.com/user/ai-native-dotnet-bootstrap).
```
